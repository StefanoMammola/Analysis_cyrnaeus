ISD-2022-0045, Isaia, Mammola & Arnedo

Phylogenetic data matrices and trees

maximum likelihood analyses (IQTREE)
-cors_16_xs.fasta: 16S sequence matrix, fasta format-cors_18_xs.fasta 18S sequence matrix, fasta format-cors_28_xs.fasta 28S sequence matrix, fasta format
-cors_c1_xs.fasta COI sequence matrix, fasta format-cors_h3_xs.fasta H3 sequence matrix, fasta format
-cors_best_scheme.txt: full partition scheme for IQTREE analyses
-BestPFmodel_cors.treefile preferred tree of the IQtree analyses with ultrafastbootstrap supports

parsimony analyses (TNT)
-cors_all_xs_sic.tnt: concatenated sequence matrix in TNT format with gaps scored as absence/presence data
-tnt_dataset_NSSxl_nelsen.tre: strict consensus tree of 42 most parsimonious trees-tnt_dataset_jak_24dec22.tre: jackknife tree in NEXUS format

Bayesian tree inference (MrBayes)
-cors_all_mb.nex: concatenated sequence matrix in NEXUS format with MrBayes block
-cors_all_mb.con.tre: majority rule consensus of the posterior distribution of trees combined form two chains after burnin 

Bayesian time estimation (BEAST)
cors_all_rev_01.xml: XML file for BEAST analysescors_all_rev_123.tre: maximum clade credibility tree using node height of the BEASt analysis
